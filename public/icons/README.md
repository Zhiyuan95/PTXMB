# PWA Icon Placeholder

This is a placeholder file. Please replace the icons in this directory with actual app icons.

## Required Icons:

1. **icon-192x192.png** (192x192 pixels)
   - Standard Android icon
2. **icon-512x512.png** (512x512 pixels)
   - High-resolution icon for splash screen

3. **apple-touch-icon.png** (180x180 pixels)
   - iOS home screen icon

## Design Guidelines:

- **Symbol**: Golden lotus flower or dharma wheel
- **Background**: Warm gradient from saffron orange (#b45309) to golden amber (#d97706)
- **Style**: Modern, minimalist, peaceful
- **Format**: PNG with transparency

## How to Generate:

You can use online tools like:

- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- Or design custom icons in Figma/Photoshop

Upload your 512x512 master icon and these tools will generate all required sizes.
